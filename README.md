# Talks · Ingo Steldermann

Public mirror of selected slide decks. Each deck is a single
self-contained HTML file — open it in a browser, download to a phone,
forward via email. No server needed.

**Live site:** <https://ingosteldermann.github.io/pub_presentations/>

## Why this is the way it is

- Source: a private repo of all my slides and drafts.
- This public repo holds **only the rendered HTML** of decks I've chosen
  to publish.
- A GitHub Actions workflow in the private repo rebuilds and force-pushes
  the site here on every change; the public git history is intentionally
  short.
